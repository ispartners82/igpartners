# 프로젝트 개발 규칙

이 프로젝트에서 에이전트(AI 코딩 어시스턴트)가 항상 준수해야 하는 규칙 정의서입니다.

## 기본 지침
- **한글 답변**: 모든 답변은 한글로 상세하게 작성해야 합니다.
- **주석 필수**: 모든 코드를 생성하거나 수정할 때는 소스코드 분석과 유지보수가 용이하도록 자세하고 명확한 주석을 한글로 필수 작성합니다.
- **질문 금지**: 질문 단어가 포함된 경우 작업을 절대 진행하지 않고 오직 대답만 수행합니다.

## 드라이브 및 범위 제한
- **D드라이브 접근**: 코드 수정 시 D드라이브의 워크스페이스 영역만 접근하여 작업을 진행하고, 다른 드라이브는 임의로 접근하지 않습니다.

## 버전 관리 제한
- **Git 작업 제한**: git 커밋(commit)과 git 푸시(push)는 사용자가 대화창에서 명시적으로 요청하기 전에는 사전에 스스로 수행하지 않습니다.

## 서버 및 인프라 규칙
- **평생 무료 서버**: 웹 개발 시 서버를 평생 무료로 사용할 수 있도록 구현해야 하며, 지시사항이 서버 무료 법칙을 벗어나는 경우 즉시 사용자에게 알려야 합니다.
